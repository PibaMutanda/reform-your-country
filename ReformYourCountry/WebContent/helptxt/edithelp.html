Aide BBCode


1.[Quote], référence bibliographique

Référencer un livre:

[quote bib="abreviation"]ma citation[/quote]

Affichera le texte "ma citation" ,sur lequel le deplacement de la souris créera une pop up qui référencera les details du livre renseigné par l'attribut "bib", bib étant l'abréviation du livre. Le titre du livre, le nom de l'auteur et l'année de publication seront affichés sous le texte "ma citation".

Reférencer un livre dans le texte:

L'auteur dit: [quote bib="abreviation" inline="true"]ma citation[/quote]

Affichera juste "ma citation" au sein du texte courant, sur lequel le deplacement de la souris créera une pop up qui affichera les details du livre renseigné par l'attribut "bib". L'attribut "inline", avec la valeur "true", permet de voir "ma citation" au coeur du texte autours ("L'auteur dit:" dans notre exemple) et sans la ligne avec les détails sur le livre.

Réferencer un auteur:

[quote author="Anonyme" ]Sors les deux vaches qu'on les compte[/quote]

Affichera "Sors les deux vaches qu'on les compte". Le texte sera suivi du nom de l'auteur qui, ici est "Anonyme".

Réferencer un auteur avec un lien web externe:

[quote author="Anonyme" out="http://www.site.com"]Sors les deux vaches qu'on les compte[/quote]

Affichera "Sors les deux vaches qu'on les compte". Le texte sera suivi du nom de l'auteur, sur lequel le lecteur pourra cliquer. L'attribut out (optionnel) permet de spécifier le site vers lequel diriger le visiteur(ne pas oublier le "http://"). C'est en général le site d'où est extrait la citation.

[unquote]

Zaratoustra a dit: [quote]Il y en a 2 [unquote](roues)[/unquote] devant et 2 derrière.[/quote]

Le tag unquote permet de revenir au style (font et couleur, mais pas taille) du texte normal hors des balises [quote]. Dans l’exemple ci-dessus, les mots Zaratoustra et roues sont dans la même font.

[Untranslated]

[quote]Je suis une citation [untranslated]I’m a quote[/untranslated][/quote]

Le tag untranslated peut être mis dans un quote, avant ou après le texte. Celui-ci contient la version originale du texte qui se trouve dans le reste du tag quote. A l'interieur d'un tag untranslated, on ne peut pas avoir d'autre tags, comme [link].

Attributs
Nom Usage Type Description
bib optionel alphanumérique renseigne l'abreviation du livre existant dans la base de donnée
inline optionel "true" or "false" si true , la citation sera affichée tel quel
author optionel si "bib" pas utilisé alphabétique renseigne le nom d'un auteur d'une citation, le nom sera affiché sous la citation, ne doit pas être utilisé conjointement de "bib"
out optionel si "author" est utilisé url affiche un lien html, ne peut-être utilisé que si l'attribut "author" est renseigné

2.[Link]

Référencer un lien:

[Link article="abcd"]un article[/link]

Affichera un lien html vers un article enregistré sur enseignement2.be. Dans notre exemple, "abcd" est le nom symbolique-raccourci de l'article.

Référencer un lien dans une citation:

[quote]Je suis une citation avec [link article="devoirs"]un hyperlien[/link] vers un autre article dedans[/quote]

Affichera une citation avec un lien html vers l'article "devoirs" à l'intérieur

Attributs
Nom Usage Type Description
article optionel alphanumérique renseigne l'URI d'un article du site enseignement2.be
out optionel URL renseigne un lien web vers lequel le lecteur sera redirigé
abrev optionel alphabétique abréviation d'un article d'enseignement2.be

4.[Image]

Référencer une image:

class="exemple">[image name="monImage.jpg" style="float:right;"/]

Affichera une image uploadée. L'attribut "name" identifie l'image.
Optionnellement, on peut indiquer un attribut style qui sera transcrit tel quel dans l'html. C'est typiquement utilisé pour positionner l'image dans le texte à gauche ou a droite, avec un padding.

5.[Escape]
Inclure des brackets
[escape] Je fais des maths [ comme ceci [ ou cela ]]]]]]]]]]. Point. [/escape]

Permet d'include des brackets "[" ou "]" dans le texte sans qu'ils ne soient interprétés comme des débuts ou des fins de balises.
